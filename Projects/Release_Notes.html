Release Notes for X-CUBE-BLE2 Projects

Copyright © 2022 STMicroelectronics

Purpose

The X-CUBE-BLE2 Projects provide examples and applications to aid understanding how to develop an application using the BLE Middleware for the BlueNRG-2 device. Projects include STM32CubeMX project file (.ioc) for the graphical visualization of the STM32 microcontroller pins, peripherals and middlewares configuration.
